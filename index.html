Esta página web está diseñada para ayudarte a organizar y centralizar tus proyectos y tareas de programación durante 16 semanas. Aquí encontrarás un resumen de cada semana con enlaces directos a la documentación y a los repositorios de GitHub de tus trabajos.


¿Qué es GitHub?

GitHub es una plataforma en línea que permite almacenar, gestionar y colaborar en proyectos de programación. Es muy utilizada por desarrolladores y empresas de todo el mundo para:

  • Guardar código fuente en la nube usando Git para el control de versiones.
  • Colaborar con otros programadores en proyectos compartidos.
  • Documentar y organizar el desarrollo de software.
  • Crear portafolios de proyectos para mostrar tus habilidades.

¿Para qué sirve esta página?

Esta página es tu espacio personal para organizar las tareas semanales que realices en tus estudios o proyectos personales. Está estructurada de forma sencilla:

  • Inicio: Información general sobre la página y sus beneficios.
  • Semanas: Cada semana contiene un resumen de tus dos tareas principales, con enlaces a documentación y repositorios.
  • Accesos rápidos: Desde aquí puedes abrir directamente tus tareas en GitHub sin buscar entre carpetas.

Beneficios de usar esta página

  • Mantener todo tu material ordenado en un solo lugar.
  • Ahorrar tiempo al buscar código o documentación.
  • Desarrollar una rutina de organización y aprendizaje.
  • Mejorar tu portafolio con enlaces directos a GitHub.

Semana 1

Tarea 1: Crear un programa que calcule el precio final de una compra aplicando un descuento doble del 7% + 7%, mostrando los cálculos de cada paso.
Tarea 2: Diseñar un programa que, dado un monto total de inversión, calcule cuánto se destina a cada rubro (feria, publicidad, transporte, etc.) según porcentajes establecidos.

Semana 2

Tarea 1: Crear un programa que determine el importe de la compra, el importe del descuento, el importe a pagar y la cantidad de lapiceros de obsequio mediante el ingreso del precio de la docena y la cantidad de docenas
Tarea 2: determine el sueldo básico, el sueldo bruto, el descuento y el sueldo neto mediante el ingreso de las horas trabajadas, la categoría para la tarifa y adicionalmente el numero de hijos para la bonificación
